# Automated-Sentiment-Based-Consumer-Support-System-Using-Machine-Learning
In this project, we built an ABSA pipeline. A keyword-driven aspect extraction module that identifies aspects from the reviews and the review text is then converted into aspect conditioned inputs. These inputs train multiple models, including TF-IDF + SGD-SVM, TF-IDF + Logistic Regression, a CNN, DistilBERT, and RoBERTa.
